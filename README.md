# 🔴⚪️ Mărțișor Digital 2026

An interactive collectible card experience built for 1st March — 
the Romanian spring tradition of gifting mărțișoare for good luck.

## What it is
12 pixel art collectible cards with rarity tiers (Common → Legendary), 
a box-opening reveal mechanic, and two Ultra Rare cards that glow gold on unlock. 
Built and shipped in under a week using AI-assisted development.

## Built with
HTML · CSS · JavaScript · Canvas API · Claude
